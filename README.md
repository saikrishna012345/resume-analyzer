AI Resume Analyzer

AI Resume Analyzer is a web-based application that automates resume evaluation using artificial intelligence. It is designed to help job seekers improve their resumes by providing structured insights, keyword analysis, and actionable feedback.

This project demonstrates practical implementation of full-stack development, file processing, and AI integration.

Key Features
-> Upload and process resumes in PDF format
-> Automatic text extraction using PyPDF2
-> Fast and lightweight Flask-based backend
-> AI-driven resume analysis (in progress)
-> Designed for future enhancements like scoring, keyword matching, and feedback generation

Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Libraries: PyPDF2
Concepts Used: File handling, text extraction, API development, AI integration (planned)

How to Run
-> Backend
  cd backend  
  python app.py  

-> Frontend
Open the frontend/index.html file in your browser to access the application.
